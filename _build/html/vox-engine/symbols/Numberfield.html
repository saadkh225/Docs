5.7. Numberfield (data)¶

METHODS FOR IMPLEMENTING NUMBERFIELDS

Numberfield¶
Numberfield.setValue(value)¶
Arguments:
  • value (number) – The numerical value to be assigned

Assigns value to the numberfield

var x = 10;
inputs('Q').setValue(x);
Returns:none
Numberfield.number(none)¶
Arguments:
  • none –

Returns the value entered

var valueEntered;
valueEntered =  inputs('Q').number();
Returns:number
